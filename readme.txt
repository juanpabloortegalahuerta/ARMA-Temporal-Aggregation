This repository contains functions that carry out the temporal aggregation of ARMA models and compute the forecasting error formulas explained in detail in the papers (referred as paper 1 and paper 2 in the code documentation)

Grigoryeva, L. and Ortega, J.-P. [2014] Hybrid forecasting with estimated temporally aggregated linear processes. To appear in the Journal of Forecasting. http://papers.ssrn.com/sol3/papers.cfm?abstract_id=2148895

Grigoryeva, L. and Ortega, J.-P. [2014] Asymptotic forecasting error evaluation for estimated temporally aggregated linear processes. To appear in the International Journal of Computational Economics and Econometrics.
